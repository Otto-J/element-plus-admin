# element-plus-admin

## 愿景 Wish

想做一个顺手的 vue3+element-plus 的 admin。紧跟 element-plus 最新版。
Always the element-plus latest version of admin project.

## 定位 Targeting

- 提供 template 模板，也就是 shim 瘦身版
- 保留必要依赖
- vite@^latest
- element-plus@latest 最佳实践
- vue@^3.2.0 + setup
